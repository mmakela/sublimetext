# Sublime Text 2 Configuration

## Installation

### OSX

    $ cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages
    $ git clone https://github.com/mmakela/sublimetext.git User

### Linux

    $ cd ~/.config/sublime-text-2/Packages
    $ git clone https://github.com/mmakela/sublimetext.git User
